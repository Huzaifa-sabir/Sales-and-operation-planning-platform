üîß TROUBLESHOOTING
================================================================================

MongoDB won't start?
  ‚Üí Check if port 27017 is free
  ‚Üí Run: netstat -ano | findstr ":27017"

Backend won't start?
  ‚Üí Check if port 8000 is free
  ‚Üí Check if Python venv exists

Frontend won't start?
  ‚Üí Delete node_modules\.vite folder
  ‚Üí Run: npm install

Need to restart?
  1. Run: stop-sop-portal.bat
  2. Wait 5 seconds
  3. Run: start-sop-portal.bat
